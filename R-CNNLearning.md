# R-CNN：区域卷积神经网络

- 论文 [Rich feature hierarchies for accurate object detection and semantic segmentation](https://arxiv.org/abs/1311.2524) 提出将**区域建议**和**CNN**结合，构建新的网络，叫R-CNN
  - **Region Proposals：区域建议** -> 产生图像建议（区域）
    - selective search
  - **CNN** -> 产生特征向量
    - AlexNet
  - 分类器 -> 预测目标类别
    - SVM
